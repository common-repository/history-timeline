Plugin for display an history timeline based on tags.

== Description ==

Plugin for displaying an history timeline based on tags, now it support Before Christ dates.
Choose an input date format for discriminate the tags for build the timeline, for each tag this plugin extract a random post from all related posts.
You can also choose the output format of the date and the timeline order.
The css is customizable from the admin area of the plugin.
Compatible with qtranslate

== Installation ==

If you have never use this plugin or a previous version, you have extract the archive file (.zip) and upload the folder history-timeline in wp-content/plugins of your web space. Than activate the plugin in plugin menu of admin control panel.

Choose your input date format for discriminate tags in the configuration page of plugin.
Than put **[history_timeline]** where you want to view the timeline.
